🚗 Arduino 4WD Smart Robot Car

A 4-wheel drive (4WD) Arduino-based robotic vehicle capable of autonomous navigation using sensors. This project demonstrates embedded systems, robotics, motor control, and sensor integration using Arduino.

📌 Features

* 🚗 4WD robotic platform
* ⚡ Arduino-based control system
* 🔋 Rechargeable battery powered
* 🎯 Obstacle detection
* 📏 Ultrasonic distance sensing
* 🎮 Motor speed and direction control
* 🔄 Autonomous navigation
* 🛠 Modular design for future upgrades

🛠 Hardware Used

* Arduino Uno
* L298N Motor Driver
* HC-SR04 Ultrasonic Sensor
* 4 DC Geared Motors
* 4 Robot Wheels
* Acrylic Chassis
* 18650 Battery Pack
* Breadboard
* Jumper Wires
* Power Switch
* IR Sensor
* IR Remote

💻 Software

* Arduino IDE
* Embedded C / Arduino C++

⚙️ Working

1. Power on the robot.
2. Arduino continuously reads data from the ultrasonic sensor.
3. If an obstacle is detected:
    * Stop
    * Change direction
    * Continue moving
4. Motors are controlled through the L298N driver.
